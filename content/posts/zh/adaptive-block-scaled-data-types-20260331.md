# Adaptive Block-Scaled Data Types

> 来源: [arXiv](https://arxiv.org/abs/2603.28765v1)

作者: Jack Cook, Hyemin S. Lee, Kathryn Le 等

NVFP4 has grown increasingly popular as a 4-bit format for quantizing large language models due to its hardware support and its ability to retain useful information with relatively few bits per parameter. However, the format is not without limitations: recent work has shown that NVFP4 suffers from its error distribution, resulting in large amounts of quantization error on near-maximal values in each group of 16 values.
