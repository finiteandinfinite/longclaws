# The Token Is a Group Element: On Lie-Algebra Attention over Matrix Lie Groups

> 来源: [arXiv](https://arxiv.org/abs/2606.20547v1)

作者: Przemyslaw Musialski

We place the attention token on the group: a token is an element $g_i$ of a matrix Lie group $G$ -- a bare transformation, with no feature payload and no external action $ρ(g)$ carrying it. To our knowledge this is the first attention construction whose tokens are bare matrix Lie group elements: their score is the closed-form algebra norm of the relative pose rather than a learned kernel, and it reaches the affine full-frame groups that every irrep- or surjective-exp-based method must exclude.
