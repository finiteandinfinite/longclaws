# Tokenisation via Convex Relaxations

> 来源: [arXiv](https://arxiv.org/abs/2605.22821v1)

作者: Jan Tempus, Philip Whittington, Craig W. Schmidt 等

Tokenisation is an integral part of the current NLP pipeline. Current tokenisation algorithms such as BPE and Unigram are greedy algorithms -- they make locally optimal decisions without considering the resulting vocabulary as a whole. We instead formulate tokeniser construction as a linear program and solve it using convex optimisation tools, yielding a new algorithm we call ConvexTok.
