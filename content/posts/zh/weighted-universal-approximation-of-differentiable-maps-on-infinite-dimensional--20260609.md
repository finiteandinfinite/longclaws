# Weighted universal approximation of differentiable maps on infinite-dimensional manifolds

> 来源: [arXiv](https://arxiv.org/abs/2606.09820v1)

作者: Philipp Schmocker, Josef Teichmann

We generalize the universal approximation theorem for functional input neural networks (FNN) to differentiable maps by including the approximation of the derivatives. A FNN maps the input from a possibly infinite-dimensional weighted manifold to the real-valued hidden layer, on which a non-linear scalar activation function is applied, and then returns the output into a Banach space via some linear readouts.
