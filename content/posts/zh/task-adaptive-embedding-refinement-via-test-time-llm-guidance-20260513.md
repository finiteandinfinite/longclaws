# Task-Adaptive Embedding Refinement via Test-time LLM Guidance

> 来源: [arXiv](https://arxiv.org/abs/2605.12487v1)

作者: Ariel Gera, Shir Ashury-Tahan, Gal Bloch 等

We explore the effectiveness of an LLM-guided query refinement paradigm for extending the usability of embedding models to challenging zero-shot search and classification tasks. Our approach refines the embedding representation of a user query using feedback from a generative LLM on a small set of documents, enabling embeddings to adapt in real time to the target task. We conduct extensive experiments with state-of-the-art text embedding models across a diverse set of challenging search and clas
