# MeanFlow Meets Control: Scaling Sampled-Data Control for Swarms

> 来源: [arXiv](https://arxiv.org/abs/2603.20189v1)

作者: Anqi Dong, Yongxin Chen, Karl H. Johansson 等

Steering large-scale swarms in only a few control updates is challenging because real systems operate in sampled-data form: control inputs are updated intermittently and applied over finite intervals. In this regime, the natural object is not an instantaneous velocity field, but a finite-window control quantity that captures the system response over each sampling interval. Inspired by MeanFlow, we introduce a control-space learning framework for swarm steering under linear time-invariant dynamic
