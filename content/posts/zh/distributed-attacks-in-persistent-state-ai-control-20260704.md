# Distributed Attacks in Persistent-State AI Control

> 来源: [arXiv](https://arxiv.org/abs/2607.02514v1)

作者: Josh Hills, Ida Caspary, Asa Cooper Stickland

As AI coding agents become more autonomous, they increasingly ship code iteratively, with the codebase persisting across sessions. This persistence creates a new attack surface: a misaligned or prompt-injected agent can distribute attacks across pull requests (PRs) and time its payload for the PR with the best natural cover.
