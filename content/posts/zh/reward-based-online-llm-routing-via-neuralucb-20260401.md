# Reward-Based Online LLM Routing via NeuralUCB

> 来源: [arXiv](https://arxiv.org/abs/2603.30035v1)

作者: Ming-Hua Tsai, Phat Tran

This study investigates the use of NeuralUCB for cost-aware large language model (LLM) routing. Existing routing approaches can be broadly grouped into supervised routing methods and partial-feedback methods, each with different tradeoffs in efficiency and adaptivity. We implement a NeuralUCB-based routing policy and evaluate it on RouterBench under a simulated online setting.
